# Getting Started with Personal Area

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-index).

## Instruction for run this project.

First step install packages for frontend/mock-api

```
yarn install

yarn --cwd ./mock-api install
```

Last step just run this project

```
yarn start
```
